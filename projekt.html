Beschreibung

In unserem WDP Projekt wurde ein Online Quiz mit JavaScript umgesetzt.

Grundsätzlich ist unser Quiz an die ORF-Show "Q1 - Ein Hinweis ist falsch" angelehnt.

Das heißt, es wird mit Hilfe von vier Hinweisen ein Begriff gesucht, wobei einer der Hinweise falsch ist bzw. nicht zum gesuchten Begriff gehört.

Weiters gibt es einen Joker "Blende falschen Hinweis aus", welcher, wie schon der Name besagt, den falschen Hinweis ausschließt.

Beantwortet der Spieler eine Frage richtig, so gelangt er zur nächsten Frage. Die herunterlaufenden Punkte werden dann zu seinem Score hinzu addiert.

Somit gilt, je schneller ein Spieler die Frage beantwortet, je mehr Punkte bekommt er zu seinem Score hinzu addiert.

Beantwortet der Spieler eine Frage falsch, so ist das Spiel beendet.
